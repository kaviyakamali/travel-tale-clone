# Wanderly - Travel Booking Platform

## Project Overview
Wanderly is a modern travel booking platform inspired by Airbnb. Users can explore destinations, search stays, browse categories, and save properties to their wishlist.

## Features
- Responsive Design
- Search Destinations
- Property Categories
- Wishlist Functionality
- User Authentication
- Mobile Friendly Interface

## Technologies Used
- React.js
- TypeScript
- Tailwind CSS
- Vite
